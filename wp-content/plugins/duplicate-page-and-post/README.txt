Duplicate a Post or Page with just one click.

== Description ==

Duplicate Page and Post does exactly what it says it will. No need to build styling and layout from scratch, with this plugin you can copy any page or post to a new draft for further editing with just one click, including styling, content and title!

= New Features =
* Duplicate multiple posts/pages via Bulk Actions
* Duplicate using the Admin Bar
* Duplicate straight from the Edit Post/Page view
* Now works seamlessly with many third-party plugins

= How To Use =
1. Go to 'All Posts' or 'All Pages'
2. Hover over the post/page you wish to duplicate
3. Click the 'Duplicate' link

= Compatible With / Supports =
* All In One SEO Pack
* bbPress
* Divi Builder
* Page Builder by SiteOrigin (requires SiteOrigin Widgets Bundle)
* WooCommerce
* WPBakery Visual Composer
* Yoast SEO
If you would like any other plugin to be supported, please raise a support topic.

== Installation ==

= From your WordPress dashboard =
1. Visit 'Plugins > Add New'.
2. Search for 'Duplicate Page and Post'.
3. Activate Duplicate Page and Post from your Plugins page.

= From WordPress.org =
1. Download Duplicate Page and Post.
2. Upload the 'duplicate-page-and-post' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...).
3. Activate Duplicate Page and Post from your Plugins page.
